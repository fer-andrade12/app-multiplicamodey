# Assets visuais do README

Coloque aqui os arquivos usados na seção **Visual do app** do `README.md`:

- `ux-flow.gif`
- `clientes.png`
- `simulacao.png`
- `investimentos.png`
- `dashboard.png`

## Padrão recomendado

- Resolução: 1366x768 ou 1440x900
- GIF: 12–20 segundos, 10–15 fps
- Tamanho do GIF: idealmente abaixo de 12 MB

## Sugestão rápida para criar o GIF

1. Abra o app em `http://localhost:3000`.
2. Grave o fluxo principal com ScreenToGif/OBS.
3. Exporte como `ux-flow.gif`.
4. Copie o arquivo para esta pasta.
